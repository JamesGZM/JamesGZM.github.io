Apache HTTP 客户端弃用

在 Android 6.0 时，就已经取消了对 Apache HTTP 客户端的支持。 从 Android 9.0 开始，默认情况下该库已从 bootclasspath 中移除。但是耐不住有些SDK中还在使用，比如我见到的友盟QQ分享报错问题。
